# JustCode Homebrew Tap

Homebrew tap for [JustCode](https://github.com/kingeke/justcode) — a lean, transparent terminal coding assistant where *you* control every token.

## Install

```sh
brew tap kingeke/justcode
brew trust kingeke/justcode   # third-party taps are untrusted by default
brew install justcode
```

Then run `justcode`.
